# SpringProjectLibrary


The program imitates the work of a digital library.

### Technology stack:
 -Java 17    
 -Spring Framework (MVC)   
 -Spring Data JPA  
 -Apache Maven  
 -Tomcat  
 -PostgreSQL  
 -JDBCTemplate  
 -Hibernate     
 -Thymeleaf  
 -Lombok  
 -Docker  
 
 ## Deploy application
 **Prerequisites**
 
 1. Java 17+
 2. Apache Maven
 3. Docker
 
 ### Build
 ```mvn clean install```  
 ```docker build -t spring_app .```  
 ```docker-compose up --build```  
 ### Open
 ```http://localhost:8080/SpringProjectLibrary/home```
